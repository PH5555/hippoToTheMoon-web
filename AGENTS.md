# 프로젝트 개요

- 프로젝트 명: hippoToTheMoon-server
- 목표: 한국투자증권 Open API를 연동하여 실시간 주식 데이터를 시각화 하고, 사용자가 가상 자산으로 주식 거래를 경험할 수 있는 서비스 구축.

주요 기능:

- 소셜 로그인: 카카오, 구글 (OAuth 2.0).
- 실시간 시세 및 차트: WebSocket 통합을 통한 시장 데이터 스트리밍.
- 모의 투자: 가상 화폐를 이용한 매수, 매도, 취소 주문.
- 실시간 랭킹 시스템: 사용자 수익률 추적 및 리더보드.

# 기술 스택

- 언어: typescript
- 프레임워크: React
- API호출: React Query
- 상태관리: zustand
- UI 라이브러리: shadcn/ui
- 외부 API: 카카오 로그인, 구글 로그인

> 이후 구현에 필요한 라이브러리는 추가 가능

# 폴더 구조
- components: 공통 UI
- hooks: 재사용 가능한 커스텀 훅
- api: API 호출 및 데이터 패칭 로직
- store: 상태 관리 전역 설정
- types: 공통 인터페이스/타입
- utils: 공통 유틸리티 함수
- screen: UI

# 프로젝트 원칙
- API 문서를 첨부하면 해당 API에 맞는 디자인과 로직을 코드로 구현할 것
- 디자인을 할 때는 ui-design 룰을 참고할 것
- 성능상 이슈가 없도록 렌더링 최적화를 할 것
